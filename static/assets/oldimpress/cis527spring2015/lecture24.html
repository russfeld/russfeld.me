Cloud Computing

  • Originated in "time sharing" in the 1950ss
  • Early network diagrams used a cloud for unknown networks
  • Businesses started using VPNs in the 1990s
  • Popularized in 2006 by Amazon's Elastic Compute Cloud (EC2)
