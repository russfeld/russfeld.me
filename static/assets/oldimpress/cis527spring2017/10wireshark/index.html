Wireshark

  • Packet capturing software
  • Can sniff and analyze packets on networks
  • It understands packet structure & encapsulation
  • Originally called Ethereal

WARNING

  • DO NOT USE ON K-STATE'S NETWORK!
  • Wireshark allows monitoring of network traffic, which is a violation of K-State's policies
  • Use only within your VM network!

Installation

  • https://www.wireshark.org/download.html
  • sudo apt-get install wireshark
