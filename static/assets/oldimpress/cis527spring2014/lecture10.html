Simple Network Management Protocol (SNMP)

  • Interface for querying inforamtion from network devices
  • Can also be used to modify settings
  • Commonly used as a monitoring tool to gather information from a variety of devices and servers

Simple Network Management Protocol

SNMP MIB and OID

  • MIB = Management Information Base
  • OID = Object ID
  • SNMP requests and responses use a coded system to identify information
  • 1.3.6.1.2.1.5.1 = icmpInMsgs
  • http://www.alvestrand.no/objectid/

SNMP Activity

  • Log on to CIS Linux
  • Use SNMP to query the number of ICMP messages sent
  • Use SNMP to query the number of SNMP messages received
  • Use SNMP to find the interfaces available on the system
  • Use SNMP to find the IP addresses of the system
