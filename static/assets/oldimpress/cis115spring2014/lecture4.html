Augustus De Morgan

De Morgan's Law


Negation (inverse) of a logic statement


¬ ( A ∧ B ) = ( ¬ A ) ∨ ( ¬ B)

¬ ( A ∨ B ) = ( ¬ A ) ∧ ( ¬ B)


Distribute the negative (¬) then swap ands (∧) and ors (∨)

Boolean Algebra

  • ∨ works like addition ( + )
  • ¬ works like negation ( − )
  • ∧ works like multiplication ( × )
  • Associative: (A ∧ B) ∧ C = A ∧ (B ∧ C)
  • Commutative: (A ∧ B) = (B ∧ A)
  • Distributive:
    A ∧ (B ∨ C) = (A ∧ B) ∨ (A ∧ C)
