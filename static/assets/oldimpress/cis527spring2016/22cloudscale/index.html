CIS 527

Advanced Cloud Management

Scaling

  • Scale your servers based on load or need
  • Can be done manually or automatically
  • Many scripting tools available

Digital Ocean

  • DOProxy Ruby Script
  • Ruby library for managing droplets
  • DigitalOcean Metadata Service
  • Configurable Proxy such as HAProxy
  • UserData for configuration

Image Source: DigitalOcean

Droplet Metadata

  • Web Interface for accessing information about a droplet
  • Useful for scripting & management

  • curl http://169.254.169.254/metadata/v1/

AWS

  • Automated scaling via configurable rules
  • Specify policies based on metrics
  • Can scale up or down automatically

Image Source: AWS

OpenStack

  • Large suite of Open Source cloud management tools
  • Covers many different use cases
  • Many parts available directly from OS repositories
  • Some parts are more mature than others

Image Source: OpenStack

Ganeti

  • Scripts in Python for controlling virtual machines
  • Very fault-tolerant design
  • Originated at Google, now maintained at the Oregon State University Open Source Lab

Image Source: Lance Albertson
