Application Servers Overview

Application Server

Image Source: Wikipedia

Application Server

  • Provide a platform for running applications
  • Perform back-end processing and storage
  • Always on, always available
  • Backbone of most organizations

Common Application Servers

  • Web - Apache, IIS, ngnix, Tomcat
  • Database - MySQL, Microsoft SQL, PostgreSQL
  • Email - Exchange, Postfix, Exim
  • Customer Relationship Management Software
  • Inventory & Accounting Software

Web Servers

  • First created in 1989 by Tim Berners-Lee
  • Provides web pages via HTTP
  • Modern examples provide scripting support for dynamic webpages, virtual hosting, caching, etc

Apache

  • 53% market share (May '13)
  • Key player in initial WWW growth
  • Cross platform (commonly Unix-like)
  • Many modules available

Internet Information Services (IIS)

  • 17% market share (May '13)
  • Microsoft's proprietary web server (NT)
  • Partially caused US v. Microsoft case
  • Some extensions available
  • Natively supports ASP.NET

nginx

  • 15% market share (May '13)
  • Reverse proxy web server
  • Runs on Unix-like systems
  • High performance & concurrency
  • ~2.5 MB memory per 10k inactive connections

Database Servers

  • aka Database Management Systems (DBMSs)
  • Manages data storage definitions, updates, retrievals, and administration
  • May store data in files or on block devices
  • Supports some query language

Microsoft SQL Server

  • 68% of companies use MSSQL
  • Stand-alone software for Windows
  • Uses T-SQL for querying
  • Commonly used with IIS for web applications

MySQL / MariaDB

  • 50% of companies use MySQL
  • Open source DBMS
  • Cross-platform
  • Now owned by Oracle
  • Typically used as part of the LAMP stack for web applications

PostgreSQL

  • Object-relational DBMS
  • Focuses on extensibility and standards compliance
  • Cross-platform
  • Used by Yahoo, State Farm, Disqus, etc.

Email Server

  • aka Message Transfer Agent
  • Handles transfer and delivery of email messages
  • Supports POP3, IMAP, SMTP and other protocols

Exim

  • Open source, cross-platform MTA
  • Default MTA on many Linux systems
  • Used in many high-traffic systems
  • 47% market share (2013)

Postfix

  • Open source, cross-platform MTA
  • Developed as a replacement for Sendmail
  • Designed for high performance delivery (~2500/sec)
  • 26% market share (2013)

Exchange

  • Microsoft's email, calendar and contact management server
  • First used in 1995 by Microsoft itself
  • 8% market share (2013)

Other Concerns

  • Frameworks - Zend, Wiki, Drupal
  • Platforms - Rails, Tomcat, MS Dynamics
