Email

  • Protocol for sending messages across the internet
  • Used as early as the 1970's as an extension to FTP
  • Originally ASCII, extended by Multipurpose Internet Mail Extensions (MIME)

Email Procotols

  • Post Office Protocol (POP3)
  • Simple Mail Transfer Protocol (SMTP)
  • Internet Message Access Protocol (IMAP)
  • Exchange ActiveSync
